Documentacion agenda_app con pydoc

    Instrucciones:
Para iniciar el programa, se debe ejecutar el archivo controlador.py
Se lanzara la aplicacion de la agenda en la cual se puede agregar, modificar, borrar y buscar los contactos que se registren.
El controlador dispara una ventana de tkinter la cual va a tomar su configuracion grafica desde el archivo vista.py
Aqui se encuentra el codigo de visual.
Desde la vista, se hace llamar a la clase Crud que esta localizada en el archivo modelo.py, aca se desarrollo todas las funciones logicas del programa.
Tambien aqui se realiza la conexion a la base de datos en sqlite3.
